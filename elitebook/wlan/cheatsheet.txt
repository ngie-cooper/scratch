Release notes are here: http://www.broadcom.com/docs/linux_sta/README.txt
Tarballs are linked here: http://www.broadcom.com/support/802.11/linux_sta.php

Prereqs:

yum install -y gcc kernel-devel

1. Download the debian package:

http://packages.ubuntu.com/lucid/all/broadcom-sta-source/download

2. Extract the debian package:

dpkg --ignore-depends=bzip2,quilt,make,debhelper --unpack broadcom*.deb

3. Patch the sources:

$ diff -u src/include/linuxver.h.orig src/include/linuxver.h
--- src/include/linuxver.h.orig 2013-10-10 16:31:42.755726084 -0700
+++ src/include/linuxver.h      2013-10-10 16:23:37.959892136 -0700
@@ -20,7 +20,11 @@
 #if (LINUX_VERSION_CODE < KERNEL_VERSION(2, 6, 0))
 #include <linux/config.h>
 #else
+#if LINUX_VERSION_CODE < KERNEL_VERSION(2, 6, 33)
 #include <linux/autoconf.h>
+#else
+#include <generic/autoconf.h>
+#endif
 #endif
 #include <linux/module.h>

4. Run make:

export M=$(pwd)/modules/broadcom-sta/amd64
export src=/usr/src/kernels/`uname -r`
make -C $src M=$M src=$src clean
make -C $src M=$M src=$src
sudo make -C $src M=$M src=$src modules_install
